## 📌 About the Project

This project was built as part of a front-end web development exercise. The goal was to design a visually appealing and brand-consistent website for **Keith Wilson**, a fashion label, using only core web technologies — no frameworks, no JavaScript.

The website reflects the elegance and style of the brand through thoughtful use of typography, color, spacing, and layout.

---

## ✨ Features

- 🏠 **Home / Hero Section** — Eye-catching landing area with brand tagline
- 👔 **Collections Section** — Showcasing featured fashion items
- 🎨 **Brand Identity** — Consistent color palette and typography across all pages
- 📱 **Responsive Design** — Adapts to different screen sizes (desktop, tablet, mobile)
- 🧭 **Navigation Bar** — Smooth and clean top navigation
- 📞 **Contact / Footer Section** — Brand contact details and social links
